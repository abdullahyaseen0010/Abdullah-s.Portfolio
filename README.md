# Abdullah's Portfolio

A personal portfolio website built with **Next.js 15** (App Router, Turbopack), **React 19**, and **TypeScript**, styled with **Tailwind CSS 4** and animated using **Framer Motion**. Icons are handled with **lucide-react**.

## Sections

- **Navbar** – responsive navigation with a mobile menu and dark/light theme switcher
- **Hero** – landing intro section
- **Skills** – scrolling marquee showcase of skills
- **Projects** – featured project cards with tech-stack tags and GitHub/demo links
- **Services** – grid of offered services
- **Testimonials**
- **Contact** – contact form with info cards
- **Footer** – socials, links, and copyright

> ⚠️ **Note:** The projects currently shown in the Projects section (e.g. "AI Task Manager," "Real-Time Chat," "E-Commerce Analytics," "Blockchain Wallet," "Weather Forecast," "Code Snippets") are **placeholder content**. Their GitHub and demo links point to generic URLs (`github.com`, `demo.com`) and are not real, deployed projects they exist to demonstrate the layout of the projects grid and will be replaced with actual project data.

## Tech Stack

Next.js · React · TypeScript · Tailwind CSS · Framer Motion · lucide-react

## Getting Started

First, run the development server:

```bash
npm i
# then
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
